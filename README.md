# IoT-Based-Multi-Sensor-Data-Acquisition-System
This project was developed as part of a two-month Makernova program focused on embedded systems and sensor integration. The objective was to design and implement a real-time multi-sensor data acquisition system using ESP32 to monitor environmental and motion parameters.

System Implementation:
The system integrates multiple sensors with ESP32 programmed using Arduino IDE. Sensor data is acquired through structured communication protocols and processed for real-time monitoring via a locally hosted interface.

Sensors Integrated:
BNO085 – Orientation and motion sensing
MQ135 – Air quality monitoring
BMP280 – Temperature and pressure measurement
Ultrasonic sensor – Distance measurement
TCS34725 – Color detection

Embedded Concepts Applied:
GPIO configuration and sensor interfacing
I2C, SPI, and ADC communication
ADC/DAC fundamentals
Real-time data acquisition
Localhost-based data visualization

Development & Simulation Tools:
Arduino IDE (ESP32 programming)
Wokwi (Initial ESP32 simulation)
Proteus (Circuit design and validation)

Key Learning Outcomes:
Multi-sensor integration on a single microcontroller
Practical implementation of communication protocols
Real-time embedded system design
Sensor calibration and data handling
